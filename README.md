# stateful-js-app

Build a JS App using a state management library, organise modules in distinct folders; with each state represented by its respective feature module. The root module would contain the app root state.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/stateful-js-app.git
cd stateful-js-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
